# Tandemloop Screening Test - Kalyan Dakkili

## Language Used "Java"

---------------------------------------------------

## Problem 1: Calculator

**Description:**  
A small calculator using a class that performs **Addition, Subtraction, Multiplication, and Division**.  

**Input:**  
- `a` (double)  
- `b` (double)  
- `operation` (string: add, subtract, multiply, divide)  

**Output:** Result of the operation.  

**File:** `Problem1_Calculator.java`

---------------------------------------------------

## Problem 2: Generate Odd Numbers

**Description:**  
Given a single integer `n`, generate the first `n` **odd numbers**.  

**Input:** Single integer `n`  
**Output:** List of first `n` odd numbers  

**File:** `Problem2_OddNumbers.java`

---------------------------------------------------

## Problem 3: Patterned Odd Series

**Description:**  
Given a single integer `n`, generate a series of odd numbers following this pattern:  

- If `n` is odd → series length = `n`  
- If `n` is even → series length = `n-1`  

**Input:** Single integer `n`  
**Output:** Series of odd numbers  

**File:** `Problem3_OddSeries.java`

---------------------------------------------------

## Problem 4: Count Multiples

**Description:**  
Count the number of integers in a list that are multiples of numbers from **1 to 9**.  

**Input:** Space-separated integers  
**Output:** Number of multiples for each number from 1 to 9  

**File:** `Problem4_MultiplesCounter.java`

---------------------------------------------------

## How to Run

1. Open the project folder in terminal/PowerShell.


2. Compile each program:

  - javac Problem1_Calculator.java

  - javac Problem2_OddNumbers.java

  - javac Problem3_OddSeries.java
    
  - javac Problem4_MultiplesCounter.java

3. To Run each Program:
   
   - java Problem1_Calculator
   
   - java Problem2_OddNumbers
     
   - java Problem3_OddSeries
     
   - java Problem4_MultiplesCounter
